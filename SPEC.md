# 五子棋游戏规格文档

## 1. 项目概述
- **项目名称**: 华丽五子棋 (Gomoku Deluxe)
- **类型**: 网页游戏
- **核心功能**: 双人对战五子棋游戏，支持悔棋、重新开始
- **目标用户**: 休闲玩家

## 2. UI/UX 规格

### 布局结构
- 全屏布局，垂直居中
- 左侧：游戏信息面板 (200px)
- 中间：棋盘区域 (自适应)
- 右侧：控制面板 (200px)

### 视觉设计

#### 色彩方案
- 背景: 深色渐变 `#0a0a1a` → `#1a1a2e` → `#16213e`
- 棋盘: 木质纹理效果 `#c4a35a` / `#8b7355`
- 黑子: 渐变 `#1a1a1a` → `#4a4a4a`，带光泽
- 白子: 渐变 `#f5f5f5` → `#e0e0e0`，带光泽
- 强调色: 金色 `#ffd700`，青色 `#00d4ff`
- 文字: `#ffffff`，`#cccccc`

#### 字体
- 主标题: "Cinzel Decorative", serif (Google Fonts)
- 正文: "Noto Sans SC", sans-serif

#### 视觉效果
- 棋盘：CSS 3D变换，阴影效果
- 棋子：径向渐变 + box-shadow 营造立体感
- 落子：缩放动画 (0 → 1) + 波纹效果
- 获胜连线：高亮发光效果
- 背景：动态粒子效果/星空

### 组件

#### 棋盘
- 15x15 网格
- 格子大小: 40px
- 边框: 8px 实线，木质颜色
- 阴影: 多层投影营造悬浮感

#### 棋子
- 直径: 34px
- 黑子: 黑曜石质感
- 白子: 珍珠质感
- 最后一手: 红色标记

#### 按钮
- 悬停: 发光效果 + 缩放
- 点击: 波纹动画
- 样式: 玻璃拟态 (Glassmorphism)

#### 信息面板
- 显示当前玩家
- 回合数
- 获胜信息
- 悔棋按钮
- 重新开始按钮

## 3. 功能规格

### 核心功能
1. 双人对战 (黑白交替)
2. 落子检测 (只能在空位)
3. 胜负判定 (横、竖、斜向五子连线)
4. 悔棋功能 (可回退一步)
5. 重新开始
6. 高亮最后一步
7. 高亮获胜五子

### 用户交互
- 点击棋盘落子
- 点击按钮执行操作
- 悬停预览落子位置

### 动画效果
- 落子: 0.3s 缩放弹跳
- 获胜: 连线闪烁发光
- 按钮: 0.2s 悬停效果
- 背景粒子: 持续飘动
